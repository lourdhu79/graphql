// to know process runnin and kill those task
On Windows: Open Command Prompt as an administrator and run:
netstat -ano | findstr :4000
This will show you the Process ID (PID) of the process using that port. Then, use the Task Manager or the taskkill command to terminate it:
taskkill /PID <PID> /F
